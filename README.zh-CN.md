## 用法💨

[发布](https://github.com/HidemaruOwO/ReScaler/releases/latest)或者[行动](https://github.com/HidemaruOwO/ReScaler/actions/workflows/build.yml)下载二进制文件并运行它。

## 安装😊

请运行这个脚本。

```bash
yarn install
```

## 构建🔨

```bash
yarn tauri build
```
