## 사용법 💨

[풀어 주다](https://github.com/HidemaruOwO/ReScaler/releases/latest)또는[행위](https://github.com/HidemaruOwO/ReScaler/actions/workflows/build.yml)에서 바이너리를 다운로드하고 실행하십시오.

## 설치 😊

이 스크립트를 실행하십시오.

```bash
yarn install
```

## 빌드🔨

```bash
yarn tauri build
```
